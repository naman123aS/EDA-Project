# Exploratory Data Analysis (EDA) Project

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a dataset to identify patterns, relationships, and trends using statistical summaries and visualizations.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

## Features

- Load and analyze dataset
- Generate statistical summaries
- Identify trends and patterns
- Correlation analysis
- Data visualization

## Visualizations Used

- Histogram
- Boxplot
- Bar Chart
- Correlation Heatmap

## Project Structure

```text
EDA-Project/
│
├── EDA.py
├── sales_data.csv
├── README.md
```

## Installation

```bash
pip install pandas matplotlib seaborn
```

## Run Project

```bash
python EDA.py
```
